# net-ssdp-discover
Discovers devices on the local network using SSDP (Simple Service Discovery Protocol) and reports their service types and locations. - Focused on Basic network operations

## Install
`git clone https://github.com/ShadowStrikeHQ/net-ssdp-discover`

## Usage
`./net-ssdp-discover [params]`

## Parameters
- `-h`: Show help message and exit
- `-st`: No description provided
- `-mx`: No description provided
- `-t`: No description provided
- `-r`: Number of retries for sending the SSDP discovery message. Defaults to 3.
- `-v`: Enable verbose output.
